UNIHUB-University Student Collaboration Platform
->The UniHub project is a web-based platform designed to connect university students and provide a centralized system for sharing academic resources. The main objective of the platform is to enhance collaboration among students and make study materials easily accessible.
->The platform includes features such as user authentication, discussion forums, resource sharing, and event updates. Students can log in, create posts, interact with others, and access study materials easily.
->The main objective of UniHub is to improve communication and collaboration among students, making it easier to exchange knowledge and stay informed. The system is designed with a simple and user-friendly interface to ensure accessibility for all users.
